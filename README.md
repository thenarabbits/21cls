# 21cls
Official repository for **"21st Century Love Story"** by Athena Chang and Kaylee Dinh.

- We made this visual novel for Milkyway, a [Hack Club](https://hackclub.com/) program
- Special thanks to [@themagicfrog](https://github.com/themagicfrog) and Skech for making "You or Me" ([link to game](https://themagicfrog.itch.io/you-or-me)), which helped inspire 21CLS :)
- Background resources credit: [Uncle Mugen](https://alte.itch.io/uncle-mugens-backgrounds)
